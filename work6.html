Process

I started with the title. It was necessary to make it loud so that it would immediately attract attention.This one was chosen because I think that mathematics lessons in schools are very boring and not interesting, and a teacher with a different presentation of information is needed.

Further, in order to interest the user, I used an attractive USP. It works when scrolling the page and the numbers are updated.It is important that you need to write those advantages that other competitor sites do not have.

I thought for a long time how best to make courses so that it would not be boring, but there was a lot of information. I decided that if a person is interested, then he will look; if not, then it will be closed. This block is beautiful and contains a lot of information.

The site has a map so that it would be easy to get there, but I made it more convenient, you can already build a route on the spot and already go along it. It is very comfortable.

It was necessary to place a video on the site. When I embed a YouTube video with its video player, I realized that this is not suitable - YouTube restricts user interaction, I created my video player using JS.

It was necessary to place a form and a block of contacts on the "Get Started" button. I think that it turned out very nice and complete.

Complex Block

Difficulty

: The most difficult thing here is to do not just one test, but to do 3 tests with different questions that would show the correct answers at the end. And another difficulty was that the questions should be random and if there are 10 of them, then more than 30 are needed so that they do not repeat.

Decision

:I solved it with the help of classes. For each new test, I created a class and it automatically took all the parameters. In order for the program to understand which test the user opened, I used classes. On each click, a class is applied. And I solved a large number of issues using JSON.
