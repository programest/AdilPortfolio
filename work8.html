Introduction

About Work

I did this work for  freelance. TThis work was like a diploma thesis at the university. This application allows you to choose Shawarma in the city and order it

A  thesis  and  presentation  were written for this application



It was necessary to implement an application that would be both for   users  and for  couriers  who would accept orders.

I think I did it on 8 out of 10.

Process

The application needed to implement a search by location. And also mark it so that users could conveniently choose by type of dish.

It will also be a huge advantage if the application has a map with a search for the place you need.

It is also worth talking about how the database is implemented in my application. For the database, I used Firebase

The application is available not only for users but also for couriers. Therefore, the functionality for couriers is different.

There are only 3 points that the courier works on. Picked up the order -> Reached the place -> Gave the order

For security and completion of the order, authorization by PIN code was introduced. When the courier presses the last 3rd item, a window pops up in which he must enter the PIN code that the user will have

Complex Block

Difficult: The most difficult part of the application was implementing user interactions with the courier.

Decision: This was implemented using Firebase when a guest orders something, a block is created in the database cell and it is shown to the courier. And all the information about the order is transmitted
